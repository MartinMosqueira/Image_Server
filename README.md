# Image Server Project

## Starting 🚀

- Cloning Project:
```
https://github.com/MartinMosqueira/Image_Server.git
```

### Pre-requisites 📋

- Python 3.10.12
- Docker 25.0.3

### Installation 🔧

- Docker Images:
```
- docker pull redis:7.2.4
```
- Run redis container:
```
- docker run -d --name redis_serverImages -p 6379:6379 redis:7.2.4
```
- Run project:
```
- python3 main.py
```
